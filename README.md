# Physics-Animation-Simulation
A real-time interactive physics simulation built using Python, NumPy, and Matplotlib. It models particle motion with gravity, collisions, friction, and obstacle interactions inside a bounded 2D environment.
A real-time interactive physics engine built with Python, simulating particle motion, gravity, collisions, friction, and obstacle interactions using **NumPy** and **Matplotlib animation tools**.

---

## 🚀 Overview

This project simulates a 2D particle system where 100 particles move freely inside a bounded environment, interacting with:

- 🌍 Gravity
- 💥 Elastic collisions (walls & particles)
- 🧱 Static obstacles
- 🌫️ Frictional damping
- 🎛️ Real-time parameter control (sliders)

It is designed as both a **visual physics sandbox** and a **learning tool for computational simulation concepts**.

---

## 🎯 Key Features

- Real-time particle motion animation
- Adjustable physics parameters:
  - Gravity
  - Elasticity (bounciness)
  - Friction
- Particle-particle collision system (simplified physics model)
- Obstacle collision handling
- Boundary wall reflections
- Dynamic color variation for visualization
- Reset functionality for fresh simulations

---

## 🧠 Physics Concepts Implemented

- Newtonian motion (F = ma simplified)
- Conservation of momentum (approximate)
- Elastic & inelastic collision modeling
- Vector-based velocity decomposition
- Frictional decay model
- Boundary constraint handling
